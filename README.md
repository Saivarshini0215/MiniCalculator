# Mini Calculator (Core Java)

This is a console-based calculator developed in Core Java.  
It demonstrates OOP concepts, exception handling, and menu-driven programming.  

## Features

- Addition, Subtraction, Multiplication, Division
- Percentage, Power, Square Root
- Input validation with error handling
- Keeps calculation history

1. Open terminal/command prompt in `src/` folder.
